# Sustainable-Materials-and-Solvent-Analytics-Using-Python
Overview

This repository contains two data-driven projects focused on sustainability assessment in chemistry and materials science. Using Python, scientific databases, and data analytics techniques, the projects demonstrate how computational methods can support sustainable decision-making in chemical processes and advanced materials development.

Project 1: Data-Driven Evaluation of Sustainable Solvents

This project analyzes commonly used organic solvents using molecular descriptors obtained from the PubChem database. Key sustainability indicators such as polarity, solubility, environmental persistence, and toxicity were derived from molecular properties to compare conventional solvents with greener alternatives.

Key Features
Automated data collection from the PubChem API
Molecular descriptor analysis (Molecular Weight, XLogP)
Sustainability scoring framework
Green solvent classification
Comparative case study of ethanol and benzene
Data visualization and interpretation
Technologies
Python
Pandas
Requests
Matplotlib
PubChem AP
